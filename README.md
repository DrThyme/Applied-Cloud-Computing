# Applied-Cloud-Computing

USAGE
```bash
* Use "python genesis.py" to start the instance
* SSH into the server and run "celery worker -l info -A app.celery" inside the "Applied-Cloud-Computing" folder.
* SSH into the server from another terminal and run "python app.py" inside the "Applied-Cloud-Computing" folder.
* Navigate your browser to "http://<floating ip>:5000/run" and wait for the results to appear. This will take some time so please be patient.
```
